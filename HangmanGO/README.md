# HangmanGo - Go Implementation

This is a rewrite of the Hangman game in Go (Golang), inspired by the original Python project.

## Features

- Modular code structure
- ASCII art for hangman states
- Word list management
- User profile support (optional)
- Idiomatic Go practices

## Getting Started

1. Install Go: https://golang.org/dl/
2. Build and run:
   ```bash
   cd HangmanGO
   go run main.go
   ```

## Project Structure

- `main.go` - Entry point
- `asciiart/` - ASCII art for hangman states
- `game/` - Game logic
- `words/` - Word list management
- `profiles/` - (Optional) User profile management

---

This project is a full rewrite and does not depend on the original Python code.
